# PWAN Real Estate Landing Page

A modern, responsive landing page for PWAN Real Estate's Prosperity Heights Estate Phase One in Ibusa, Delta State.

## Features

- ✅ Responsive design
- ✅ Countdown timer
- ✅ Interactive forms
- ✅ Modern UI with Tailwind CSS
- ✅ Mobile-friendly
- ✅ Lead capture forms

## Technologies Used

- React.js
- Tailwind CSS
- Lucide React Icons
